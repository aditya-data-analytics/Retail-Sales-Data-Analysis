# Retail Sales Data Analysis using Excel

## Project Overview

This project analyzes retail sales performance using Microsoft Excel.
The objective of the analysis is to identify sales trends, evaluate store and department performance, and generate actionable business insights that can support strategic decision-making in the retail industry.

---

## Dataset Source

Kaggle Retail Dataset
https://www.kaggle.com/datasets/manjeetsingh/retaildataset

The dataset consists of three main files:

• Sales dataset
• Stores dataset
• Features dataset

These datasets were combined to create a unified dataset for analysis.

## Key Business Questions

• Which stores generate the highest revenue?
• Which departments contribute the most to total sales?
• Do holiday weeks significantly increase sales?
• Are there seasonal patterns in retail demand?
• How do external factors influence sales performance?

---

## Key Insights

• Large stores generate higher overall revenue.
• A small number of departments contribute the majority of total sales (Pareto principle).
• Holiday periods significantly increase retail sales.
• Strong seasonal patterns appear during November and December.
• External economic factors have smaller influence compared to seasonal demand and promotions.
